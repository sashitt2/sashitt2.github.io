Cancer Phylogenetics

Paction

Paction is a method to reconcile SNV and CNA clone proportions and trees for thesame tumor. The output is a set of tumor clones characterized by both SNVs and CNAs,the clone proportions and a comprehensive clone tree annotated with both SNV andCNA events. [Code]

Doublet Detection

doubletD

doubletD is an efficient doublet detector designed for scDNA-seq data. IncorporatingdoubletD in scDNA-seq analysis pipelines leads to more accurate results. [Code]

Transcript Assembly

Jumper

Jumper is a code that takes a bam file of RNA-seq reads and a Fasta file of the reference genome as input and reconstructs RNA transcripts and their abundances in the sample. [Code]

Phylodynamics

TiTUS

TiTUS is a code that takes as input a timed pathogen phylogeny with leaves labelled by the host along with epidemiological data such as entry-removal times for the hosts and a contact map. It counts and uniformly samples from the set of feasbile interval vertex labelings of the timed phylogeny that satisfy the direct transmission constraint while supporting a weak transmission bottleneck. [Code]

  • Sashittal, P. & El-Kebir, M., Sampling and Summarizing Transmission Trees with Multi-strain Infections, Bioinformatics, 2020. [Paper]

SharpTNI

SharpTNI is a tool for counting and sampling solutions from the space of parsimonious transmission networks under a weak transmission bottleneck constraint. This problem arises in phylodynamic and phylogeographic analyses of disease outbreaks. [Code]

  • Sashittal, P. & El-Kebir, M., SharpTNI: Counting and Sampling Parsimonious Transmission Networks under a Weak Bottleneck, BMC Medical Genomics, 2020. [Paper]

Data-driven Dynamics Modeling

lrDMD

Low-rank dynamic mode decomposition (lrDMD) solves for a rank-constrained linear representation of the dynamical system for given series of data snapshots. [Code]

  • Sashittal, P. & Bodony, D., Reduced-Order Control using Low-Rank Dynamic Mode Decomposition, Theoretical and Computational Fluid Dynamics, 2019. [Paper]

KF-DMD

A data-driven method for optimal sensor placement and parameter optimization using transfer operator based reduced order models. [Code]

  • Sashittal, P. & Bodony, D., Data-driven sensor placement for fluid flows, Under review for Theoretical and Computational Fluid Dynamics, 2021.
